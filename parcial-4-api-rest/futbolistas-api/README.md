## Descripción

Esta evaluación parcial consiste en una API REST desarrollada en PHP puro, sin frameworks, para administrar futbolistas de un equipo de fútbol.

La API permite realizar las operaciones básicas de un CRUD:

- Crear futbolistas
- Consultar todos los futbolistas
- Consultar futbolista por ID
- Actualizar futbolistas
- Eliminar futbolistas

## Tecnologías utilizadas

- PHP 8+
- MySQL
- PDO
- JSON
- XAMPP
- Postman
