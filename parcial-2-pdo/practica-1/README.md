## Descripción

Esta práctica consiste en desarrollar una aplicación web CRUD para la gestión de productos utilizando PHP, PDO, programación orientada a objetos y MySQL.

El sistema permite registrar, listar, editar y eliminar productos desde una sola página web.

## Tecnologías utilizadas

- PHP 8+
- MySQL
- PDO
- XAMPP
- Bootstrap 5
- Programación Orientada a Objetos
