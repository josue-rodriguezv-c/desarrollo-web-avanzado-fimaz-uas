# Práctica 1 – Parcial 1

## Objetivo

Aplicar los fundamentos de la Programación Orientada a Objetos en PHP mediante la creación de una clase con atributos privados, constructor, getters y setters.

## Descripción de la clase creada

Se creó una clase llamada Usuario con los atributos privados nombre y correo. La clase incluye un constructor para inicializar los datos, así como métodos getters y setters para acceder y modificar la información aplicando encapsulamiento.

## Instrucciones de ejecución

1. Iniciar Apache desde XAMPP.
2. Colocar el proyecto dentro de htdocs.
3. Abrir el navegador.
4. Acceder a la ruta correspondiente.
5. Verificar que se muestren los datos del usuario.

## Evidencia de funcionamiento

La aplicación muestra correctamente el nombre y correo del usuario utilizando los métodos getters.
